Welcome to the CampusCollaborate REST API documentation!

Our API provides a platform for students to collaborate on academic projects by enabling them to search for and connect with other students with the required skills. With our API, students can create and manage their profiles, list their skills, and search for other students based on their requirements.

Our API is secured using JWT token-based authentication. Explore our API documentation to learn more about our endpoints and how to use them in your application.

Features

  • Search for other students based on their skills and requirements
  • Create and manage your own profile
  • List your skills for other students to find you
  • Secure API using JWT token-based authentication
